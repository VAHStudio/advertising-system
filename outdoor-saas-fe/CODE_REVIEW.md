## 🔍 Vue 3 迁移代码审查报告

### 严重问题 (Critical) - ✅ 已修复

#### 1. ✅ `usePagination.ts` - 响应式丢失
**问题：** 使用 getter 返回解构值，在 Vue 模板中会导致响应性丢失  
**修复：** 改为返回 readonly ref，保留响应式

#### 2. ✅ `Plans.vue` - 功能不完整
**问题：** 只有骨架代码，缺少看板/列表视图完整实现  
**修复：** 
- 添加完整的看板视图 (KanbanView.vue)
- 添加列表视图 (ListView.vue)
- 添加方案详情视图 (PlanDetailView.vue)
- 实现状态映射和数据显示

#### 3. ✅ `Pagination.vue` - 类型错误
**问题：** 页码类型推断错误  
**修复：** 使用类型断言和类型保护

#### 4. ✅ `App.vue` - 缺少初始化
**问题：** App.vue 没有调用 authStore.initAuth()  
**修复：** 添加 onMounted 钩子调用初始化
